# Elena Fun Travel Website

Static GitHub Pages site.

## Upload to GitHub

Upload these files/folders to the repository root:

- `index.html`
- `assets/elena-profile.jpg`

The live site should be:

https://elenafuntravel.github.io

If the site is not live, make sure the repository name is exactly:

`elenafuntravel.github.io`

and GitHub Pages is enabled from `main` branch, root folder.
